# ApplyPilot

ApplyPilot is an AI-powered internship application agent that helps students analyze job descriptions, structure application data, and prioritize next actions.

## Problem

Students often apply to many internships across different platforms. Job descriptions are unstructured, requirements are inconsistent, and it is difficult to track deadlines, fit, and follow-up actions.

## Solution

ApplyPilot uses Gemini and Google Cloud to analyze job descriptions, extract structured information, calculate a match score based on the user's profile, and store application data in MongoDB. The agent can then retrieve opportunities, compare roles, and recommend which applications to prioritize.

## Core Features

- Analyze internship job descriptions
- Extract required skills, location, company, and role information
- Calculate a match score
- Store application records in MongoDB
- Display saved opportunities in a dashboard
- Generate next-action recommendations
- Generate tailored application notes or cover letter drafts

## Tech Stack

- React
- FastAPI
- Gemini
- Google Cloud
- MongoDB
- MongoDB MCP Server
